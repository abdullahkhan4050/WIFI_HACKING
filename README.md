## ğŸ§ª The Ethical Lab Tests

We conducted two primary ethical security tests in a controlled lab environment:

### 1. Deauthentication Attack
- **Definition:** Disconnecting users by sending deauthentication frames to the network.  
- **Learning:** This test showed how sudden disruption of internet access can occur and forms the basis of more advanced attacks, including the Evil Twin attack.

### 2. Evil Twin Attack
- **Definition:** Setting up a fake WiFi network identical to our original network to simulate credential harvesting.  
- **Why it's Dangerous:** Demonstrated how easily users can connect accidentally and have credentials captured. This attack is commonly used in phishing and Man-in-the-Middle (MITM) scenarios.
